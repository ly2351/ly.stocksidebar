# 股票监控插件 (ly-stocksidebar)

## 简介

股票监控插件是一个用于 Visual Studio Code 的扩展，可以帮助用户实时监控股票行情。插件提供了股票列表视图，支持添加、删除、移动股票，并且可以将股票添加到状态栏以便随时查看。

## 功能特性

- **实时股票数据**：从 sina 财经获取实时股票数据。
- **股票列表视图**：在 VSCode 的侧边栏中显示股票列表。
- **添加股票**：通过搜索功能添加自选股票。
- **删除股票**：从股票列表中删除股票。
- **移动股票**：上移、下移、置顶、置底股票。
- **状态栏显示**：将股票添加到状态栏以便随时查看。
- **自动刷新**：支持配置自动刷新股票数据，仅在交易时间段启用。
- **新闻监控**：在输出通道中显示选股宝新闻，支持配置启用/禁用和刷新间隔。
- **手动刷新**：通过工具栏按钮手动刷新股票数据。

## 安装

1. 打开 Visual Studio Code。
2. 进入扩展市场（Extensions Marketplace）。
3. 搜索 `ly-stocksidebar`。
4. 点击安装按钮进行安装。

## 使用方法

### 添加股票

1. 打开股票监控视图（`股票监控` 侧边栏）。
2. 点击 `添加股票` 按钮。
3. 在弹出的快速选择框中输入股票代码或名称进行搜索。
4. 选择要添加的股票并确认。

### 删除股票

1. 打开股票监控视图（`股票监控` 侧边栏）。
2. 右键点击要删除的股票。
3. 选择 `删除股票`。

### 移动股票

1. 打开股票监控视图（`股票监控` 侧边栏）。
2. 右键点击要移动的股票。
3. 选择 `上移股票`、`下移股票`、`置顶股票` 或 `置底股票`。

### 添加到状态栏

1. 打开股票监控视图（`股票监控` 侧边栏）。
2. 右键点击要添加到状态栏的股票。
3. 选择 `添加到状态栏`。

### 从状态栏删除

1. 打开股票监控视图（`股票监控` 侧边栏）。
2. 右键点击要从状态栏删除的股票。
3. 选择 `从状态栏删除`。

### 手动刷新

1. 打开股票监控视图（`股票监控` 侧边栏）。
2. 点击工具栏中的刷新按钮（🔄）以手动刷新股票数据。

### 新闻监控

插件会在输出通道（`股票监控`）中显示选股宝的最新新闻，包括利多/利空信息、摘要和相关板块。新闻按时间轴反转输出（最旧的先），并只显示新的新闻。

- 打开输出通道：`查看 > 输出 > 股票监控`。
- 新闻会自动刷新（可配置），有新新闻时会显示输出通道。

## 配置选项

在 VSCode 的设置中，你可以配置以下选项：

- **ly-stocksidebar.fallColor**：下跌时的颜色，默认为 `NavajoWhite`。
- **ly-stocksidebar.riseColor**：上涨时的颜色，默认为 `LightBlue`。
- **ly-stocksidebar.stocks**：自选股票代码列表，默认为空数组。
- **ly-stocksidebar.statusBarStock**：显示在状态栏的股票代码列表，默认为空数组。
- **ly-stocksidebar.enableAutoRefresh**：启用自动刷新定时器，默认为 `true`。
- **ly-stocksidebar.refreshInterval**：自动刷新间隔（毫秒），默认为 `5000`，最小 `1000`。
- **ly-stocksidebar.enableOutputChannel**：启用输出通道新闻监控，默认为 `true`。
- **ly-stocksidebar.outputInterval**：输出通道刷新间隔（毫秒），默认为 `20000`，最小 `5000`。


## 注意事项

- 插件依赖新浪财经接口获取股票数据，请确保网络畅通。
- 交易时间段为周一至周五的 9:30 - 11:30 和 13:00 - 15:00，非交易时间段数据不会自动刷新。
- 新闻监控依赖选股宝 API，仅显示新的新闻，并按时间轴反转输出。
- 配置更改后可能需要重启 VSCode 或重新加载窗口以生效。
