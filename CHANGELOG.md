# Change Log

All notable changes to the "stocksidebar" extension will be documented in this file.

## [1.0.17] - 2025-12-30

### 优化
- CHANGELOG内容整理

## [1.0.16] - 2025-12-30

### 修改
- 优化股票百分比显示格式：现在显示为两位小数（如 +0.00% 或 -0.00%），提升显示精度和美观性。
- 更换股票图标：上涨股票使用自定义 `meat.svg` 图标，下跌股票使用 `noodles.svg` 图标，增强视觉识别。
- 重构代码结构：更新 `StockProvider` 构造函数以接收扩展上下文，支持自定义图标路径的正确加载。

## [1.0.15] - 2025-12-30

### 新增
- 添加 `parentRefresh` 和 `parentAddStock` 命令，支持从父节点刷新和添加股票。
- 更新菜单配置：在视图工具栏和项目上下文菜单中添加相关选项。
